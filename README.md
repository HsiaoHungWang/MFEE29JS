7. 使用遞迴函式求最大公因數
```
使用輾轉相除法
計算gcd(18,48)時，先將48除以18得到商2、餘數12，

即得到最大公因數6。我們只關心每次除法的餘數是否為0，為0即表示得到答案。
```
